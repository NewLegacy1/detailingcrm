# Stock images and add-on icons

Default SVG placeholders are included. Replace with your own photos or icons as needed.

## Service images (stock)

Folder: **`stock/`**

| Filename | Used for |
|----------|----------|
| `interior-detail.svg` | Interior Detail (placeholder included) |
| `exterior-wash.svg` | Exterior Wash (placeholder included) |
| `inside-out-detail.svg` | Inside and Out Detail (placeholder included) |
| `ceramic-coating.svg` | Ceramic Coating (placeholder included) |

Replace with `.jpg` or `.png` and update template `image_url` in the DB if you use a different extension.

## Add-on icons

Folder: **`icons/`**

| Filename | Used for |
|----------|----------|
| `pet-hair.svg` | Pet Hair Removal (included) |
| `sand.svg` | Sand Removal (included) |
| `carpet-shampoo.svg` | Carpet Shampoo (included) |
| `leather-conditioning.svg` | Leather Conditioning (included) |

You can customize any template in the CRM after adding from template; replacing files here updates the default for new orgs and new “Add from template” copies.
